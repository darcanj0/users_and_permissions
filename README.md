## Instalação

Clone o projeto

```bash
  git clone https://github.com/darcanj0/users_and_permissions.git
  cd ./users_and_permissions
```

Dê a permissão para rodar o script
```bash 
  sudo chmod 777 ./fullSolution.sh
```

Faça login como usuário root

```bash 
  su
```

Rode o comando
```bash
  ./fullSolution.sh
```
